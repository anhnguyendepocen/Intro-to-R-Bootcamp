What to Remember from this Section

Every ggplot2 plot has three key components:

  1. Data
  2. A set of aesthetic mappings between variables in the data and visual properties
  3. At least one layer which describes how to render each observation. Layers are usually created with a geom function.

Let's Add Geoms

To display the data we need to tell ggplot what to draw

geom_histogram() histogram
geom_freqpoly() frequency polygon
geom_bar() bar chart
geom_point() scatter plot
geom_line() line chart
geom_boxplot() boxplot


Check out all the available geoms at docs.ggplot2.org/current

Univariate Geoms

Histogram & Frequency Polygons

ggplot(data = mpg, aes(x = hwy)) +
        geom_histogram()

ggplot(data = mpg, aes(x = hwy)) +
        geom_freqpoly()

Histogram & Frequency Polygons

ggplot(data = mpg, aes(x = hwy)) +
        geom_histogram(bins = 25, color = "grey30", fill = "white")

ggplot(data = mpg, aes(x = hwy)) +
        geom_freqpoly(bins = 25, color = "blue")

Bar Chart

ggplot(data = mpg, aes(x = class)) +
        geom_bar()

ggplot(data = mpg, aes(x = cyl)) +
        geom_bar()

ggplot(data = mpg, aes(x = factor(cyl))) +
        geom_bar()
